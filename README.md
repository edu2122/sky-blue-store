# Sky Blue Shop VE

Tienda de camisetas del Mundial 2026 construida con Next.js App Router,
Tailwind v4 y shadcn/ui.

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- shadcn/ui
- Zustand (carrito)

## Requisitos

- Node.js 20+
- pnpm 10+

## Desarrollo

```bash
pnpm install
pnpm dev
```

Abrir: http://localhost:3000

## Build

```bash
pnpm build
pnpm start
```

## Lint

```bash
pnpm lint
```
